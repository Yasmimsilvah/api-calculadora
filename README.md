# API Calculadora

## Bibliotecas utilizadas

- FastAPI
- Uvicorn
- Pyngrok

## Funcionalidades

A API recebe dois números e uma operação matemática: soma, subtração, multiplicação, divisão.

## Requisição esperada

{ <br>
 "a": 10, <br>
 "b": 5, <br>
 "operacao": "soma", <br>
 "resultado": 15 <br>
} <br>
